A modern, AI-powered Netflix clone built using React.js and Tailwind CSS, enhanced with Gemini AI for smart, multilingual movie recommendations.

#FEATURES:

-> Responsive UI – Works smoothly on all screen sizes

-> AI-Based Movie Search – Search and chat with AI to find the best movies

-> Personalized Suggestions – Ask things like “Suggest some horror movies” and get accurate results

-> Multilingual Support – Browse and interact in multiple languages

-> Real-Time Movie Data – Integrated with TMDB API for up-to-date movie info

-> Firebase Authentication – Secure login and signup

-> Dynamic Trailers – Watch official movie trailers directly in the app

-> Optimized Performance – Uses custom hooks and memoization to avoid repeated API calls
